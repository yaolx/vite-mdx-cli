# <center>vscode 插件：picgo</center>

## 前言

一直都是用 obsidian 写 md 文档，本地复制图片可以直接显示，还是挺方便的，但是如果要上传到 github 上面，文件地址因为是本地的，无法正常显示。然后得去 github 上面调整，很不方便。

## 图床

搜索了几款图床工具，还是 vscode 的 picgo 方便。
只需配置好 github 的相关信息，复制好图片，然后 ctrl + alt + c，即可上传到 github 仓库中，又设置了 cdn 地址，图片加载也很快
